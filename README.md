# Orion Driver (ALSA / LINUX)

This is a custom audio driver for modified Chromebooks.

## How to install

Follow these steps to install the Orion Driver on your Chromebook:
1. Download the ZIP file from this Github repository.
2. Extract the downloaded ZIP file.
3. Open the extracted folder named Orion_driver.
4. Click on the three dots inside the folder.
5. Select "Open in console" (or Terminal).
6. Run the installation script by typing this command:
sudo ./install.sh
